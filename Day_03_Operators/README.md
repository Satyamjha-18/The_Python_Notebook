# 📅 Day 03 - Operators in Python

## 📖 Overview

Operators are special symbols used to perform operations on variables and values. Python provides different types of operators such as Arithmetic, Assignment, Comparison, Logical, Identity, Membership, and Bitwise operators.

This day covers the theory, syntax, examples, and practical programs for all Python operators.
