# 📅 Day 04 - Conditional Statements in Python

## 📖 Overview

Conditional statements allow a program to make decisions based on different conditions. They control the flow of execution by running different blocks of code depending on whether a condition is True or False.

This module covers all Python conditional statements with theory, syntax, examples, and practice programs.

---

## 📚 Topics Covered

- Introduction to Conditional Statements
- if Statement
- if-else Statement
- if-elif-else Statement
- Nested if
- Short-hand if
- Short-hand if-else (Ternary Operator)
- Logical Operators with Conditions
- Mini Projects
- Practice Questions
- Interview Questions
