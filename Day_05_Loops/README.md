# 📅 Day 05 - Loops in Python

## 📖 Overview

Loops are used to execute a block of code repeatedly until a specific condition is met. They help reduce code duplication and make programs more efficient.

This module covers Python loops from basic to advanced with practical examples and exercises.

- Why Loops are Used
- for Loop
- while Loop
- range() Function
- Nested Loops
- break Statement
- continue Statement
- pass Statement
- Infinite Loop
- Mini Programs
- Practice Questions
- Interview Questions
