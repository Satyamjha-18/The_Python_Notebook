# 📅 Day 06 - Lists in Python

## 📖 Overview

Lists are one of the most powerful and widely used data structures in Python. They allow storing multiple values in a single variable and support indexing, slicing, modification, and many built-in methods.

This module covers everything about Python Lists with theory, practical examples, and interview questions.

---

## 📚 Topics Covered

- Introduction to Lists
- Creating Lists
- List Indexing
- Negative Indexing
- List Slicing
- Updating List Elements
- Adding Elements
- Removing Elements
- List Methods
- Looping Through Lists
- Nested Lists
- List Comprehension
- Copying Lists
- Sorting Lists
- Practice Programs
- Interview Questions
