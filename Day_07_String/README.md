# 📅 Day 07 - Strings in Python

## 📖 Overview

A String is a sequence of characters enclosed in single quotes (' '), double quotes (" "), or triple quotes (''' ''' / """ """).

Strings are one of the most commonly used data types in Python and are immutable, meaning they cannot be changed after creation.

This module covers Python Strings from basic to advanced with practical examples.
