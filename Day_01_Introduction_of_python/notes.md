## 📌 What is Python?

Python is a high-level, interpreted, object-oriented, and easy-to-learn programming language created by Guido van Rossum in 1991.

It is known for its simple syntax, readability, and powerful libraries. Python is widely used in Data Science, Artificial Intelligence, Machine Learning, Web Development, Automation, Cyber Security, and Software Development.

## 📌 Comments in Python

Comments are used to explain the code, improve readability, and make the program easier to understand. Python ignores comments during execution.

### 1. Single-Line Comment

A single-line comment starts with the `#` symbol.

```python
# This is a single-line comment
print("Hello, Python!")
```

### 2. Multi-Line Comment

Python does not have a dedicated multi-line comment syntax. Developers commonly use triple quotes (`'''` or `"""`) to write multi-line descriptions or documentation.

```python
"""
This is a multi-line comment.
It can span multiple lines.
"""
print("Welcome to Python")
```

### Why Use Comments?

- Explain the purpose of the code.
- Make code easier to read and maintain.
- Help other developers understand your program.
- Temporarily disable code during testing or debugging.
